Basic HTML Tags and descriptions

Tag Name Description
<head> Head This tag should contain the metadata for the site
<h1> Heading Declares a heading, importance of the heading is indicated by the number (1 - 6)
<p> Paragraph Declares a paragraph, this is displayed in the default text on the page
<div> Division Declares a division of the html code and acts as parent to tags within.
<em> Emphasis Applies empahsis to surrounded text (default italic)
<strong> Strong Highlight surrounded text (default bold)
<span> Span Used to select or declare a short line of text
<ul> Unordered List A list declaration that uses bullet points by default
<ol> Ordered List A list declaration that uses numbered points
<li> List Object Tag that must surround each object within a list

Heading tags and Usage in HTML

You can use any of the 6 heading tags to denote a heading in html, each tag is a different size and importance in the hierarchy of the Site. h1 is the largest and most important of these tags and should only be used once per page, this will denote the most important heading on the page. Each subsequent tag is smaller than the previous (h1, h2, h3, h4, h5, h6)

Examples

<h1>This is a Main Heading!</h1>


<h3>This is a sub-heading!</h3>

Adding an image using HTML

To add an image in your HTML file, you will need to use the img tag. It will also require the following attributes.

  • src(source location of the image, this can be a local file or a link to an image online).
  • URL must be nested inside quotation marks.
  • Possible additional attributes include alt(alternate description of image)

Example

<img src="photo.jpeg" />

Adding a Video in HTML

You can use the video tag in HTML to add videos to your webpage! To do this you will need to also include a src URL, this is not a self closing tag so you will need to add a closing tag. Between the opening and closing tags you can add a message to be displayed if the video does not load.

Available Attributes

  • src -- Same as with the img tag, this is used to declare the location of the video
  • width -- Can set the width of the player
  • height -- Can set the height of the player
  • controls -- adds basic video controls to the player

Example

<video src="yourvideo.mp4" controls>Video could not be displayed</video>
