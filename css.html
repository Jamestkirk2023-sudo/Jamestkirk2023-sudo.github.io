Ruleset Terms

  • Selector: The begining of the rule set used to target the element that will be styled.
  • Declaration Block: The code in-between (and including) the curly brace that contains the CSS declarations.
  • Property: The first part of the declaration that signifies what visual characteristic of the element is to be modified.
  • Value: The second part of the declaration that signigies the value of the property.

Example of syntax

p { color:blue; }

  • p = Selector
  • area inside of {} = Declaration Block
  • color: blue; = Declaration
  • color = property
  • blue = value
